## deploy

```
// Start locally under /dev. Runs the app in the development mode.
$ hugo server -D

// Update the submodule when it is cloned again.
$ git submodule update --init --recursive

// When deploying, build under /dev
$ hugo

// Under the control of /portfolio
$ firebase deploy
```
